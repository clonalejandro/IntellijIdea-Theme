## How to use
**- Open IntelliJ IDEA**

**- Go to:** `Configure`

**- Select:** `Import Settings`

**- Select:** `clona-theme.jar`

**-Done & Restart this**

## When this has been restarted

**- Go to:** `Configure`

**- Go to:** `Preferences` or `Settings`

**- Go to:** `Editor`

**- Go to:** `Color & Fonts` 

In scheme list set:

 ```php
clonalejandro-theme
 ```
 
**- Select this:**
`apply & ok`
